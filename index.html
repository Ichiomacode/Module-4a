Differences between padding,margin and border

  1. Padding: An element's padding area is the space between its content and its border. Note: Padding creates extra space within an element
  2. Margin: creates extra space around an element. The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).
  3. CSS border:It properties allow you to specify the style, width, and color of an element's border.

In summary: padding creates space within an element, margin craetes space around an element while the border sets the border around an HTML element, meaning all four borders (top, right, bottom and left).
